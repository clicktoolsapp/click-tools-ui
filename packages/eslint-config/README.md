# ESLint config

## Whats included?

- Standard config base;
- Prettier;

<small>React Only</small>

- React plugin;
- React Hooks plugin;
- JSX a11y plugin;

## Setup

### React

Inside `.eslintrc.json`

```json
{
  "extends": "@rocketseat/eslint-config/react"
}
```

### Node.js

Inside `.eslintrc.json`

```json
{
  "extends": "@rocketseat/eslint-config/node"
}
```
